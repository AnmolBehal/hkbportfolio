01.About Me

Hello! I'm Hari, a web developer who enjoys turning complex problems into beautiful, intuitive interface designs. My journey in web development started with a curiosity about how the internet works, which quickly snowballed into a deep passion for coding and building applications from scratch.

Fast-forward to today, I specialize in building full-stack web applications using modern technologies like React, Node.js, and TypeScript. I focus heavily on writing clean, maintainable code and building highly performant, scalable APIs that power seamless front-end experiences.

Whether it's building a complex state management system, optimizing a database query, or crafting pixel-perfect UI components, I approach every task with a focus on delivering robust and user-centric solutions.
